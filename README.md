# 熊本大学 時間割シミュレーター (2025年度版)

熊本大学の学生（特に新入生）が、複雑なシラバス情報を直感的に整理し、自分だけの時間割をシミュレーションするためのWebアプリケーションです。
Vercel を使ってWeb上に公開しています:
    https://release-kmdi-timetable.vercel.app/

## 🚀 特徴
- **マルチデバイス対応**: PC/スマホのどちらでも快適に操作可能。
- **ターム制完全対応**: 前期・後期・T1〜T4の複雑な区分をフィルタリング可能。
- **プライバシー配慮**: 選択したデータはブラウザ（localStorage）にのみ保存され、サーバーへ送信されません。
- **高精細保存**: 独自CSSにより、印刷やPDF保存時にA4横サイズで美しく出力されます。
- **共有機能**: 構築した時間割をURLとして友達にシェアできます。

## 🛠 技術スタック
- **Data Engineering**: Python (REST API Scraping, Data Cleansing, Normalization)
- **Frontend**: HTML5, JavaScript (ES6+), Tailwind CSS
- **Deployment**: Vercel
- **Security**: ソースコード（スクレイパー）と公開資産をリポジトリレベルで分離。

## 📖 使い方
1. 左側の検索パネルから「学部」や「ターム」を選択。
2. 科目名や教員名で講義を検索。
3. カードをクリックして時間割に追加（重複時は赤く警告）。
4. 「保存/印刷」ボタンからPDFとして書き出し。
